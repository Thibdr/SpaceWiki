About Us

This application is created as part of a developer training project with Javascript and Angular. The 3 students working on this project are Thibault Durand, Fabien Vauclin and Céline Hénichard. The purpose of this application is to facilitate access to NASA information about the planets of the solar system. This information is intended for both novices and the most informed of our visitors. We used the data available in the NASA API "https://api.nasa.gov/index.html". We hope you will enjoy discovering our work and remain available via the contact page for any constructive suggestion that will allow us to go further in our visitors' research. The SpaceWiki team.
